## 1. 프로젝트 개요

### 1.1 비즈니스 목표
멍크린은 **반려동물 전문 청소업체**로, 반려동물과 함께 생활하는 가정 및 상가를 대상으로 전문 청소 서비스를 제공합니다. 본 홈페이지는 멍크린의 서비스를 소개하고, 고객이 간편하게 견적을 문의하며, 온라인 결제까지 가능한 **원스톱 플랫폼**을 목표로 합니다.

### 1.2 핵심 가치
- 🐾 **반려동물 전문성** — 일반 청소가 아닌 반려동물 환경에 특화된 전문 청소
- 🧪 **친환경 & 안전** — 독일 키엘사 친환경 세제, 반려동물에게 무해한 저자극 세정제 사용
- 🔬 **업계 최고 장비** — 컬비(KIRBY), 미코플러, 피톤치드, 오존발생기 등 최신 전문 장비 보유
- 👨‍🔧 **전문 인력** — 알바가 아닌 교육을 수료한 전문 인력 투입
- 📞 **연락처** — 010-2391-6070 / 010-8353-6587

### 1.3 타겟 사용자
| 사용자 유형 | 설명 |
|---|---|
| **일반 고객** | 반려동물과 함께 사는 가정 / 이사 예정자 / 출산 예정자 |
| **상가 고객** | 애견카페, 동물병원, 반려동물 관련 시설 운영자 |
| **관리자** | 멍크린 운영진 (회원 관리, 후기 관리, 견적 관리) |

### 4.2 1️⃣ 멍크린 소개 (/about)

#### 목적
멍크린의 브랜드 스토리, 차별점, 전문 장비, 전문 인력에 대한 종합 소개 페이지

#### 구성 섹션

##### (A) 브랜드 소개
- **슬로건:** "반려동물과 함께 하는 집, 청소가 달라야 합니다"
- **소개 텍스트:**
  > 멍크린은 반려동물과 함께 생활하는 공간의 특성을 깊이 이해하고, 털과 냄새 제거는 물론 위생과 공기 환경까지 고려한 반려동물 전문 청소 서비스를 제공합니다. 눈에 보이지 않는 털 하나, 남아 있는 냄새 하나까지도 세심하게 관리합니다.
- **핵심 가치 3대 원칙:**
  - 🛡️ 안전 — 사람과 반려동물 모두에게 안전한 친환경 세정
  - ✨ 전문성 — 다년간의 노하우와 최고급 전문 장비
  - 💛 신뢰 — 청소 전후 사진 공유, 투명한 서비스

##### (B) 차별점 (왜 멍크린인가?)
- ✔ 일반 가정 청소와 다른 반려동물 환경 전문 노하우
- ✔ 강아지/고양이에게 무해한 저자극 친환경 세정제 사용 (독일 키엘사)
- ✔ 냄새의 원인을 제거하는 살균 및 탈취 집중 관리
- ✔ 구역별 걸레 분리 사용 (교차 오염 방지)
- ✔ 알바가 아닌 교육 수료한 전문 인력 투입
- ✔ 청소 전/후 사진 고객 공유

##### (C) 전문 장비 소개
| 장비명 | 설명 |
|---|---|
| **컬비 (KIRBY)** | 카펫・매트・바닥 틈새 속 반려동물 털, 비듬, 미세먼지까지 강력 흡입하는 전문 장비 |
| **미코플러 (진공+스팀 동시)** | 진공청소와 스팀 청소가 동시에 가능한 바닥 전문 장비 |
| **오존발생기** | 오존(O₃)의 강한 산화력으로 세균・곰팡이・냄새를 분해하는 공기질 관리 장비 |
| **피톤치드 분무기** | 자연유래 피톤치드로 공간 전체 냄새 케어 |
| **UV 살균 장비** | 자외선 살균으로 화장실 등 세균 밀집 공간 집중 관리 |
| **고압 스팀기** | 고온 고압 스팀으로 찌든 때와 세균 동시 제거 |

##### (D) 청소 프로세스 상세
1. **오존 산화 시공** — 공기질 관리 (서랍・문 전체 개방 후 오존 가스 충진)
2. **전용 세제 도포** — 멍크린 특별 배합 친환경 세제 분사
3. **바닥 기계청소 + 스팀청소** — 미코플러로 오염물 흡입 및 스팀 살균
4. **컬비 디테일 케어** — 벽지・소파・매트리스의 미세 오염물 제거
5. **반응촉진 작업 (탈취)** — 냄새 원인 분자 중화・분해
6. **바닥 전문 장비 마무리** — 마루결을 살리며 꼼꼼한 마무리
7. **피톤치드 연무 + UV살균** — 공간 전체 피톤치드 분사 + 화장실 UV 살균
8. **나노코팅 서비스** — 주방・화장실 수전 나노코팅 (무료 서비스)

##### (E) 무료 서비스 안내
- ✅ 화장실 및 주방 전체 고압 스팀 작업
- ✅ 공간 전체 피톤치드 연무 작업
- ✅ 주방 및 화장실 수전 나노코팅 서비스

##### (F) 이런 분들에게 추천합니다
- ✔ 반려동물 냄새가 쉽게 빠지지 않는 집
- ✔ 털 알레르기, 비염으로 고민 중인 가족
- ✔ 이사를 앞두고 있는 반려동물 가정
- ✔ 출산을 앞두고 아기맞이 청소가 필요한 반려동물 가정
- ✔ 반려동물이 무지개 다리를 건넌 후 정리가 필요한 공간
- ✔ 애견카페, 동물병원 등 상가 위생 관리가 필요한 사업장

### 4.8 6️⃣ 결제하기 (/payment)

#### Toss Payments 연동

##### 결제 흐름
1. 고객이 결제 페이지 접근
2. 결제 정보 입력:
   - 서비스 유형 선택
   - 결제 금액 (관리자가 사전에 견적 확정한 금액, 또는 정가 상품 선택)
   - 주문자 정보 (이름, 전화번호, 이메일)
3. "결제하기" 버튼 클릭 → Toss Payments 결제창 호출
4. 결제 완료 → 성공/실패 콜백 처리
5. 결제 완료 페이지 표시 (주문번호, 결제 금액, 서비스 내용)

##### Toss Payments 연동 사양
| 항목 | 내용 |
|---|---|
| **PG사** | Toss Payments |
| **결제 수단** | 카드, 간편결제 (토스페이, 카카오페이, 네이버페이 등), 계좌이체, 가상계좌 |
| **연동 방식** | Toss Payments SDK (JavaScript) |
| **API 버전** | 최신 v2 API |
| **Client Key** | (추후 등록 — 테스트/운영 분리) |
| **Secret Key** | (추후 등록 — 서버사이드 보관) |
| **Webhook** | 결제 완료/취소/환불 이벤트 수신 |

##### 결제 상품 유형 (예시)
| 상품명 | 설명 | 가격 |
|---|---|---|
| 이사청소 (10평 이하) | 반려동물 전문 이사청소 | 견적에 따라 상이 |
| 거주청소 (10평 이하) | 반려동물 전문 거주청소 | 견적에 따라 상이 |
| 아기맞이청소 | 출산 전 반려동물 가정 전문 청소 | 견적에 따라 상이 |
| 무지개청소 | 사후 공간 정리 전문 청소 | 견적에 따라 상이 |
| 상가청소 | 애견카페/동물병원/시설 | 견적에 따라 상이 |

> **참고:** 대부분의 청소 서비스는 견적 기반 가격이므로, 관리자가 견적 확정 후 고객에게 결제 링크를 전달하는 방식도 고려. 또는 관리자가 결제 금액을 입력해주는 "맞춤 결제" 기능 필요.

##### 결제 관련 기능
- 결제 성공/실패 처리
- 결제 내역 조회 (고객 마이페이지)
- 환불 요청 기능 (관리자 승인 필요)
- 세금계산서 / 현금영수증 발행 연동

## 8. 비기능 요구사항

### 8.1 성능
- 메인 페이지 로딩 시간: 3초 이내
- Lighthouse 성능 점수: 85점 이상
- 이미지 최적화: WebP 포맷, lazy loading

### 8.2 보안
- HTTPS 전용
- JWT 토큰 만료 시간: Access Token 1시간, Refresh Token 7일
- SQL Injection 방지 (ORM 사용)
- XSS 방지 (입력값 sanitization)
- CSRF 보호
- 비밀번호 bcrypt 해싱 (salt rounds: 10)
- 관리자 API에 대한 역할 기반 접근 제어 (RBAC)

### 8.3 반응형 디자인
| 디바이스 | 브레이크포인트 |
|---|---|
| 모바일 | 0 ~ 768px |
| 태블릿 | 768px ~ 1024px |
| 데스크톱 | 1024px 이상 |

### 8.4 SEO
- 각 페이지별 고유 title, meta description
- Open Graph 태그 (SNS 공유 최적화)
- 시맨틱 HTML
- sitemap.xml, robots.txt
- 구조화된 데이터 (JSON-LD) — 사업장 정보

### 8.5 접근성
- WAI-ARIA 레이블
- 키보드 내비게이션
- 색상 대비 WCAG 2.1 AA 기준 충족
- alt 텍스트 (모든 이미지)
